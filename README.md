# Bash Scripting

A collection of Bash shell scripts for learning and practice.

## Scripts

| Script | Description |
|--------|-------------|
| `abc.sh` | Basic script example |
| `name.sh` | Script to work with names |
| `os1.sh` | OS detection script (version 1) |
| `os2.sh` | OS detection script (version 2) |
| `os_test.sh` | OS testing script |
| `palindrome.sh` | Check if a string/number is a palindrome |
| `primeno.sh` | Check if a number is prime |
| `sum.sh` | Calculate sum of numbers |
| `test.sh` | Test script |
| `validateDate.sh` | Validate a date input |

## How to Run

```bash
# Make the script executable
chmod +x script_name.sh

# Run it
./script_name.sh
```
